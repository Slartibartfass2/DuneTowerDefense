## Requirements

This software is using SDK 16 and Gradle 7.3.3.
It was tested on:

- Windows 10

with the resolutions and refresh rates:

- 3440x1440, 100Hz
- 1920x1080, 75Hz

Link to the used SDK: [https://jdk.java.net/java-se-ri/16](https://adoptium.net/?variant=openjdk11)
